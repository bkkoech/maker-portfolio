GAZEBO MAPPER

Description: An autonomous occupancy grid(OG) mapper for turtlebot2 simulated in gazebo using TheConstructSim
Date: Spring 2020
Location: Harvard
Software, Tools, Equipment: ROS, Python
Contributers: Billy Koech, Ernest Ochieng, Mark Peternson, Julia Ebert, Florian Berlinger

Overview

I help teach a course at Harvard on autonomous robot systems. In the spring of 2020 the occurence of the COVID-19 pendemic forced everyone to shift to remote teaching and learning. What this meant for this highly collaborative and lab heavy class is that we had to move all the assignments to a simulation environment. I therefore set up an environment in RDS studio by the TheConstructSim, designed a simulation world, and wrote a mapping program for turtblebot2 in python. The code can be found in my github.
