ERGODOX THESIS

Description: A had wired ergonomic keyboard for writing my thesis based on the ErgoDox EZ design
Date: Spring 2020
Location: Harvard
Software, Tools, Equipment: C++, Fusion 360, Soldering, Electronics, Teensy 2.0
Contributers: Billy Koech

Overview

I started developing RSI while working on the write up for my senior capstone project. Therefore, I designed, hand wired and assembled this ergonomic keyboard to reduce the strain on my hands. It is based on the ErgoDox EZ design. The top and bottom layer DXF designs can be found on the Ergodox-io repository. The mechanical keys are held in place using a 3D printed case similar to that of PrintDox. I added a palm rest to the PrintDox design as shown in the CAD design below. The firmware is written in C++ with the help of keyboard layout editor and Keyboard Firmware Builder.
